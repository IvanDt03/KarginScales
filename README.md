# Отчет о тестировании программы "Динамометрические весы Каргина"

## Описание программы
Программа моделирует работу с динамометрическими весами Каргина — измерительным устройством для термомеханического исследования полимеров. Она предназначена для лабораторного практикума по дисциплине «Высокомолекулярные соединения» для студентов специальностей «Фундаментальная и прикладная химия» и «Химия».

### Цель обновления
Ранее использовалась программа 2008 года, написанная на Objective Pascal (среда Delphi). Обновление потребовалось из-за:
- Несовместимости с Windows 10/11;
- Некорректной обработки данных из Excel-файлов;
- Примитивной визуализации, затрудняющей понимание работы прибора.

## Функциональные возможности
### Приборная панель
- **Выбор образца**: доступны полимеры (Полистирол-1, Полистирол-2, ПВХ-4, ПВХ-5, ПВХ-6, ПВХ-7) с выделенными температурными интервалами.
- **Область нагрева/охлаждения**: текущая температура отображается зеленым цветом, регулируется стрелками.
- **Кнопка "Пуск"**: моделирует изменение температуры во времени и фиксирует деформацию (Гамма).
- **Данные и измерения**: результаты автоматически сортируются по температуре.
- **График**: строится точечная диаграмма с кривыми и маркерами.
- **Сохранение данных**: результаты сохраняются при переходе между образцами.
- **Режим преподавателя**: при вводе ключ-пароля отображаются эталонные графики для проверки.

### Дополнительные улучшения
- Корректное отображение данных из Excel-файлов.
- Автоматическая сортировка данных по температуре.
- Возможность добавления/удаления образцов полимеров через Excel-файл (`Content/D.xlsx`).

## Заключение
Обновленная программа сочетает современный интерфейс, корректную работу с данными и наглядную визуализацию, что способствует лучшему усвоению материала студентами. Решение подходит для пользователей с любым уровнем компьютерной грамотности.
