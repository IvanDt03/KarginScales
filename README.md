# Динамометрические весы Каргина

**Динамометрические весы Каргина** — измерительное устройство, которое применяют для термомеханического исследования полимеров. Согласно методике построения термомеханических кривых с использованием этих весов, к образцу полимера после достижения определённой температуры прилагают нагрузку и измеряют деформацию, которая развивается в течение сравнительно небольшого интервала времени. Затем нагрузку снимают, образец нагревают до другой температуры и опыт повторяют. Проведя замеры в достаточно широкой температурной области, получают совокупность точек, по которым и строят термомеханические кривые.

Программа моделирует работу с динамометрическими весами Каргина и создана для лабораторного практикума по дисциплине «Высокомолекулярные соединения» для студентов, обучающихся на специальностях *«Фундаментальная и прикладная химия»* и *«Химия»*.

## Недостатки старой версии программы

Ранее при проведении лабораторных занятий использовалась программа 2008 года, написанная на *Objective Pascal* (среда *Delfi*). Основные проблемы старой версии:

- Недостаточно корректно работала с версиями Windows 10, 11;
- Неправильно обрабатывала результаты из Excel-файла и выдавала неточные результаты;
- Не создавала в голове студента целостный образ работы с прибором, так как визуальная составляющая достаточно примитивна.

## Возможности прибора

- **Выбор образца**: при нажатии окна «Выбор образца» появляется перечень доступных для исследования полимеров: Полистирол-1, Полистирол-2, ПВХ-4, ПВХ-5, ПВХ-6, ПВХ-7.
- **Температурный интервал**: для каждого полимера выделен температурный интервал, необходимый для работы (числа окрашены в красный цвет).
- **Область нагрева/охлаждения**: зеленым цветом отмечена текущая температура. Осуществлена возможность подъема и опускания температуры нажатиями стрелочек соответствующего направления (вверх/вниз).
- **Запуск процесса**: при нажатии кнопки «Пуск» происходит изменение температуры во времени (в окне «Текущая температура») и отображение Гаммы (деформации), которая фиксируется студентами.
- **Структурирование данных**: данные о температуре и гамме (деформации) выносятся в отдельную вкладку — «Данные и измерения» и автоматически сортируются по возрастанию (по температуре).
- **Построение графиков**: во вкладке «График» происходит построение точечной диаграммы с кривыми и маркерами для наглядного представления полученных термомеханических кривых.
- **Сохранение результатов**: при переходе к другим образцам данные о полученных результатах деформации (в таблице) и графики сохраняются.
- **Режим преподавателя**: при введении специального ключ-пароля отображаются эталонные версии графиков (из Excel-файла), на которые наносятся модельные результаты учащихся.

## Улучшения в новой версии

- Наличие графического представления динамометрических весов Каргина;
- Корректное отображение результатов в соответствии с Excel-файлом;
- Представление данных в таблице в порядке увеличения температуры (а не по хронологии измерения);
- Возможность внесения/удаления преподавателем данных о новых образцах полимера с помощью Excel-файла (в Excel-файле Content/D.xlsx в новом листе по шаблону предыдущих добавляется новый образец/для удаления необходимо просто удалить лист полимера) с автоматическим внедрением в программу (после перезагрузки программы).
